How to calculate time to break password?
To estimate the time required to break a password, you need to consider the method used by attackers, such as brute force attacks. Here’s a simplified approach to calculate the time required to break a password:

1. Determine the Search Space:
The search space is the total number of possible combinations for a given password. It can be calculated using the formula:
N = C^L
where:
N = Total number of possible combinations
C = Number of possible characters in the character set
L= Length of the password

2. Character Set Size:
The size of the character set depends on what characters are used in the password:
- Lowercase letters (a-z): 26
- Uppercase letters (A-Z): 26
- Digits (0-9): 10
- Special characters (e.g., !, @, #, $): typically around 32 (varies based on allowed characters)

For example, if a password uses lowercase and uppercase letters, digits, and special characters, the character set size is:
C = 26 + 26 + 10 + 32 = 94

3. Compute the Search Space:
For a password of length (L) :
N = 94^L
