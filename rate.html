How to determine attack rate?

Password Attempt Rates

The value of R (the rate of password attempts per second) for a normal computer can vary significantly depending on the hardware and the specific attack method used. Here's a rough estimate based on different types of computers and attack methods:

1. Average Personal Computer:

  • CPU-based brute force attack:
    • Older or lower-end CPUs: 1,000 - 10,000 attempts per second
    • Modern CPUs: 100,000 - 1,000,000 attempts per second

2. High-Performance Gaming PC:

  • GPU-based brute force attack:
    • Using a mid-range GPU (e.g., NVIDIA GTX 1660): 10,000,000 - 100,000,000 attempts per second
    • Using a high-end GPU (e.g., NVIDIA RTX 3080): 100,000,000 - 1,000,000,000 attempts per second

3. Specialized Hardware or Botnets:

  • High-end GPUs or FPGA/ASIC-based systems:
    • Specialized hardware like FPGAs or ASICs can achieve 1,000,000,000 - 10,000,000,000 attempts per second or more.
    • Large botnets comprising thousands of compromised machines can collectively achieve very high rates of password attempts.

4. Online Attacks:

  • Limited by network latency and server defenses:
    • Typically much slower due to rate limiting and lockout mechanisms: 1 - 1,000 attempts per second

For an average modern personal computer with a decent CPU, a rough estimate would be around 100,000 to 1,000,000 attempts per second. However, using a powerful GPU can increase this rate significantly.
